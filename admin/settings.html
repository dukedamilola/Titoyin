🔑 GitHub Personal Access Token
Required for automatic publishing. Stored only in your browser — never sent anywhere else.
Get your token: github.com → Settings → Developer settings → Personal access tokens (classic) → tick repo
🔐 Change Admin Password
Change your login password. Choose something strong and memorable.
After changing your password, open admin/js/auth.js in VS Code and update the ADMIN_PASSWORD value, then push to GitHub.
